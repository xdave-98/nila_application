# Project: Nila Application

## Context & Objectives
A personal, visually stunning single-page web app for Nilany ("my cat"). The app delivers a sweet, heartfelt message telling her she's loved, that her smile matters, and that "Booooo" will always try to make her happy. The goal is to create a "wow factor" experience with animations, cute visuals, and personal inside jokes.

## Architecture (high-level)
- **Single-page static HTML/CSS/JS app** — no backend needed.
- Sections scroll vertically with animated transitions.
- Sections: Hero (greeting) → Sweet message → Photo gallery (Bali, Mini Golf, cats, Kung Fu Panda) → Valentine flowers → Closing message.

## Data Model
N/A — no database. All content is static/embedded.

## Tech Stack & Dependencies
- **HTML5 / CSS3 / Vanilla JS** (no framework needed for a single page).
- **CSS animations + GSAP or AOS** for scroll-triggered animations.
- **Google Fonts** for typography.
- External image sources for cats, Kung Fu Panda, Bali, mini golf, pink flowers GIF.

## Databricks Resources
N/A — this is a purely frontend project.

## Design Guidelines
- **Colors:** Red, white, black. NO pink backgrounds — only a small pink flowers GIF.
- **Tone:** Sweet, loving, playful. Pet name: "my cat". She calls him: "Booooo".
- **Must include:**
  - Cat images (cute cats)
  - Kung Fu Panda references/images
  - Bali pictures
  - Mini Golf imagery
  - A pink flowers GIF (Valentine's Day reference)
  - Animated hearts, smooth transitions, scroll animations
- **Dark mode** as default (black background, red/white accents).

## Phases / Milestones
1. **Phase 1:** Static HTML/CSS mockup with all sections, layout, and colors.
2. **Phase 2:** Add animations (scroll reveals, floating hearts, micro-interactions).
3. **Phase 3:** Final polish, responsive design, deploy-ready.
